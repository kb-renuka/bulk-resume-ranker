# Bulk Resume Ranker

A recruiter-facing tool that ranks multiple candidate resumes against a single job description by keyword match — the bulk screening step recruiters do manually, automated.

## Features
- Paste one job description, then multiple resumes (separated by `---`)
- Instantly ranks every candidate by match score, best first
- Shows matched and missing keywords per candidate
- Fully client-side — no data uploaded anywhere

## Tech Stack
JavaScript, HTML, CSS (vanilla, no frameworks or build step)

## How to Run
Open `index.html` directly in any browser — no installation needed.

Try it live: kb-renuka-bulk-ranker.netlify.app
